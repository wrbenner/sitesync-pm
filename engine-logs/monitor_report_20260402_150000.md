## Critical Bugs Found and Fixed

### 1. Test Command Bug (HIGH IMPACT)
- **Root Cause**: Engine auto-detected test command as `npm test -- --watchAll=false`. The `--watchAll` flag is Jest syntax; vitest uses `--run`.
- **Impact**: Every cycle wasted 5 minutes on test-fix timeout. Claude Code couldn't fix it because the bug was in the test script invocation, not in code.
- **Fix**: Changed auto-detect to `npm test -- --run` in `autonomous_loop.sh` line 440.

### 2. State Reset Bug (CRITICAL — Engine Never Reached Architect Mode)
- **Root Cause**: `state.json` was unconditionally overwritten with `last_completed_cycle: 0` on every launch, including hot-reload resumes. Additionally, `ESTIMATED_SPEND` and `TOTAL_PROMPTS_EXECUTED` were reset to 0 on resume.
- **Impact**: 28 runs over 2 days, only 3 ever completed a cycle. The engine has been stuck in Surgeon mode since creation, never reaching cycle 4 (Architect mode) where it would start building P0 features like Sage Intacct integration.
- **Fix**: Added conditional to only write initial state.json on fresh runs. Restored spend and prompt counters from state.json on resume.

### 3. Score Logging Bug (MEDIUM)
- **Root Cause**: `audit_*_raw.json` files matched the `audit_*.json` glob in 4 locations. Raw files have `"module": "unknown"`, producing ghost "unknown: 0/100" entries in LEARNINGS.md.
- **Fix**: Added `[[ "$audit_file" == *_raw.json ]] && continue` skip guard to all 4 audit file iteration loops.
