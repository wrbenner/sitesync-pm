# SiteSync AI — Brand Guide

## Logo Files (in public/logos/)

| File | Use Case |
|---|---|
| `sitesync-horizontal.png` | Sidebar header (expanded), top navigation on light backgrounds |
| `sitesync-horizontal-white.png` | Sidebar header on dark backgrounds |
| `sitesync-symbol.png` | Sidebar collapsed state, mobile header, small spaces |
| `sitesync-symbol-white.png` | Dark background variant of symbol |
| `sitesync-stacked.png` | Login page, splash screens, marketing pages |
| `sitesync-stacked-white.png` | Dark background variant of stacked |

## Favicon / PWA Icons

| File | Size | Use |
|---|---|---|
| `favicon-32.png` | 32x32 | Browser tab icon |
| `icon-192.png` | 192x192 | PWA manifest icon |
| `icon-512.png` | 512x512 | PWA splash screen |

## Brand Colors

| Color | Hex | Usage |
|---|---|---|
| **SiteSync Orange** | `#F27921` | Primary brand color, CTAs, active states |
| **SiteSync Grey** | `#858484` | Logo mark, secondary text |
| Theme token: `brand400` | `#F47820` | Closest theme token to brand orange |

## Rules

1. **NEVER use the HardHat lucide icon as the logo.** Use the actual logo files.
2. **Login page:** Use `sitesync-stacked.png` centered above the form.
3. **Sidebar expanded:** Use `sitesync-horizontal.png` (or white variant on dark sidebar).
4. **Sidebar collapsed:** Use `sitesync-symbol.png`.
5. **Favicon:** Reference `favicon-32.png` in index.html.
6. **PWA icons:** Reference `icon-192.png` and `icon-512.png` in manifest.
7. **Never distort the logo.** Maintain aspect ratio.
8. **Minimum clear space:** Half the symbol height on all sides.
