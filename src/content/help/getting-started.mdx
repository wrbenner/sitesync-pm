# Getting started with SiteSync

SiteSync is the construction management platform built for general contractors who want field-grade tooling without enterprise drag. RFIs, submittals, daily logs, schedules, budgets, change orders, drawings, punch lists, and an AI copilot (IRIS) — all on a single project record, all auditable, all priced flat at $400/month or $4,080/year.

This guide gets you from signup to first project in under fifteen minutes.

## Create your first project

After signup you land on **The Day** — your role-specific command stream. To create a project:

1. Click the **+ New Project** button in the top bar.
2. Enter project name, address, project type (commercial, residential, civil, industrial), and delivery method (lump sum, GMP, cost-plus, design-build).
3. Set the project start and target completion dates.
4. Pick a cost code template (CSI MasterFormat, custom, or copy from prior project).
5. Save.

The project is now live with auto-generated RFI numbering, default cost codes, and a blank schedule ready to import.

## The 5-step onboarding wizard

On first login a wizard walks you through the critical setup steps. You can skip any step and return to it from **The Crew** or **Settings**.

1. **Verify your email.** A verification link was sent at signup. Click it to unlock invites and billing.
2. **Set up your organization.** Confirm legal name, license number (optional), default time zone, and logo.
3. **Create or import a project.** Either start fresh (above) or import from CSV (projects + cost codes).
4. **Invite your team.** Add at least one project manager and one superintendent. See [Inviting your team](./inviting-your-team).
5. **Try IRIS.** Generate a sample RFI draft from a stock prompt to see how the AI assistant fits your workflow.

Each completed step is logged to your activation funnel and visible to your account on the founder dashboard.

## Where to find things — The Nine

SiteSync organizes every project around nine top-level pages. Each is reachable from the left rail.

| Page | What lives there |
| --- | --- |
| **The Day** | Role-specific dashboard — your inbox, overdue items, today's schedule, IRIS suggestions. |
| **The Field** | Daily logs, photos, weather, manpower, check-ins, safety observations. |
| **The Plan** | Schedule (Gantt + lookahead), milestones, critical path, baseline vs actuals. |
| **The Ledger** | Budget, cost codes, change orders, invoices, lien waivers, payment applications. |
| **The Crew** | People — employees, subcontractors, vendors, roles, certifications, contacts. |
| **The Drawings** | Sheets, markups, RFI pins, revision history, OCR'd specifications. |
| **The Risk** | RFIs, submittals, punch list, safety incidents, deficiency log. |
| **The Memory** | Documents, contracts, photo library, transmittals, meeting minutes. |
| **The Audit** | Hash-chained activity log, compliance pack export, deposition-grade evidence. |

Roles are gated *per action*, not per page. Every page is reachable by every user; what they can *do* on a page depends on their role.
