# Routing an RFI for response

After you submit an RFI, it routes to the suggested answerer. Routing is a single click — no email forwarding gymnastics.

## What the answerer sees

- The full RFI body + attachments
- A reply box
- Approve / Reject / Request Clarification buttons

## Escalation

If an RFI sits unanswered past its due date, the SLA escalation ladder kicks in: gentle reminder → CC the project manager → CC the architect of record. See [SLA escalation policy](/help/sla-escalation) (coming soon).

## After response

Closed RFIs are searchable forever. The audit chain captures every state change with author + timestamp.
