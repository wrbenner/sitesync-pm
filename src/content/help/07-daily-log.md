# Logging a daily report

Daily logs are the AIA G701-style record of what happened on site each day. Required for most contracts; tedious to write longhand.

## How SiteSync helps

The daily log auto-drafts from:

- Crew check-ins (who was on site)
- Photos uploaded that day
- RFIs and submittals created or closed
- Weather forecast vs. observed conditions

You skim, edit, and approve. Usually < 90 seconds.

## Sections

The G701 standard sections are pre-populated:

1. Manpower
2. Equipment
3. Work performed
4. Issues / delays
5. Weather

Each section's source data is shown so you can verify before approving.
