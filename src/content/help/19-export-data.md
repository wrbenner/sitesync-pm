# Exporting your data

Your data is yours. We make export easy, not painful.

## Full workspace export

Settings → Workspace → **Export all data** generates a ZIP containing:

- One CSV per table (projects, RFIs, submittals, daily logs, etc.)
- One folder of all uploaded files (photos, PDFs, drawings)
- The hash-chained audit log as JSON

The export runs in the background; we email a download link when it's ready (usually under 10 minutes).

## Per-project export

From any project page, click **Export** in the header. Same shape, scoped to one project.

## API

Pro and Enterprise accounts get programmatic access via the SiteSync API. See [API documentation](/help/api) (coming soon).

## After cancellation

Export your data **before** canceling. After the 60-day grace period, the data is soft-deleted; recoverable for 30 more days, then gone.
