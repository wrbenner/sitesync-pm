# Setting up your first project

A project is the unit of work in SiteSync. Every RFI, submittal, photo, and dollar lives inside one project.

## Required fields

- **Name** — what your team calls it. Doesn't have to match the legal contract name.
- **Address** — used for weather forecasting and the project header.
- **Contract value** — anchors the budget module.
- **Start date + target completion** — anchors the schedule.

## Optional but recommended

- **Project type** — residential, commercial, civil. Affects the default RFI/submittal templates.
- **General contractor name** — only if you're the owner or sub. If you're the GC, leave blank.

## After creation

The project lands on `/dashboard`. Use the sidebar to navigate the Nine workflows. Settings → Project lets you edit any field later.
